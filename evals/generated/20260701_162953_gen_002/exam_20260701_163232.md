# 离线评测样本 测试卷

## 一、选择题

**1.** 在C语言中，关于“指针”和“指针变量”的说法，下列哪一项是正确的？
  A. 指针和指针变量是同一个概念，都表示内存地址。
  B. 定义指针变量后（如 int *p;），可以直接使用指针变量进行赋值或取值操作。
  C. 指针变量的值可以是一个任意的整数，不需要强制类型转换。
  D. 指针是程序实体所占内存空间的第一个单元的地址，而指针变量是用来存放这种地址的变量。

## 二、代码填空题

**2.** 假设有以下函数定义：

```c
int add(int a, int b) {
    return a + b;
}
```

请在下面的代码中填写函数指针的声明，使得程序可以通过指针 `p` 调用 `add` 函数并输出结果。

```c
#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    ___  // 声明一个函数指针 p，指向返回 int、参数为两个 int 的函数
    p = add;
    int result = p(3, 4);
    printf("%d", result);
    return 0;
}
```

# 参考答案

## 一、选择题

**1.** D
  > 选项A错误：指针是地址本身，指针变量是存放地址的变量，二者本质不同，不能混淆。选项B错误：指针变量定义后必须先赋值（指向一个有效地址）才能使用，否则无意义甚至引发错误。选项C错误：指针变量的值只能是指针（地址），直接赋一个普通整数而不做强制类型转换会导致编译错误或运行时错误。选项D正确：指针是内存首地址，指针变量用于存储这种地址，符合知识点定义。

## 二、代码填空题

**2.** int (*p)(int, int);
  > 本题考察函数指针的声明语法。函数指针的声明必须用括号将 `*` 和指针变量名括在一起，以表明变量是指针而非函数。正确格式为：`返回值类型 (*指针变量名)(参数类型列表);`。如果遗漏括号，如 `int *p(int, int);`，则会被编译器解释为返回 `int*` 类型的函数声明，而非指针变量。因此，空缺处应填入 `int (*p)(int, int);`，其中 `p` 是函数指针变量，可以指向任何返回 `int` 且接受两个 `int` 参数的函数。赋值时直接使用函数名（即函数地址）`p = add;` 即可，之后通过 `p(3, 4)` 间接调用 `add` 函数。
